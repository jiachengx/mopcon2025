FFmpeg 音訊處理 Pipeline

# 音量檢測
ffmpeg -i input.wav -filter:a volumedetect \
       -f null /dev/null

# 結果: mean_volume: -28.5 dB

# 語音增強 +13dB
ffmpeg -i input.wav \
       -filter:a "volume=13dB" \
       output.wav

# Ding-Dong +16dB
ffmpeg -i dingdong.wav \
       -filter:a "volume=16dB" \
       dingdong_loud.wav

# 音訊合併
ffmpeg -i dingdong.wav \
       -i silence_0.5s.wav \
       -i voice.wav \
       -filter_complex "concat=n=3:v=0:a=1" \
       final.wav

音訊處理關鍵技術

音量增益策略

語音檔: +13dB 提高嘈雜環境可聽度
提示音: +16dB 引起注意力

Ding-Dong 設計

雙音節提示音 → 0.5秒停頓 → 語音播報
總長度: 1s + 0.5s + 4~5s

音訊拼接技術

使用 concat 濾鏡無縫拼接三段音訊，確保播放連續流暢

實測效果

70dB 嘈雜環境中清晰可聞

處理速度: <100ms/片段

保持語音自然度，無明顯失真
